# Customer Ops Optimizer – Reference Data

Reference data for the **Customer Ops Optimizer** agent (software/tech companies >$500M revenue).

## Files

| File | Purpose |
|------|--------|
| `benchmarks.json` | FTE-per-revenue and role-mix benchmarks for Professional Services, Customer Success, and Customer Support; red flags for overstaffing or skewed mix. |
| `ai_agentic_levers.json` | AI/agentic automation potential by function (implementation playbooks, health scoring, tier-1 deflection, etc.) with typical scope. |
| `offshoring_levers.json` | Offshoring/nearshoring suitability by activity; what to keep onshore. |

## Customization

Edit the JSON files to align with your firm’s benchmarks or add segments (e.g. by geography or product type). The agent uses this data as context when generating assessments.
